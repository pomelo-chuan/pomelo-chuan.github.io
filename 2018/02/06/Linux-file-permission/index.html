Linux文件权限

查看文件属性

[-][rw-][rw-][r–]

第一个字符表示文件类型

  1. [d]为目录
  2. [-]为文件
  3. [l]为link file
  4. [b]为装置文件里面的可供储存的接口设备(可随机存取装置)
  5. [c]为装置文件里面的串行端口设备，例如键盘、鼠标(一次性读取装置)

剩下的分为三组:

  1. 第一组表示文件拥有者权限
  2. 同群组的权限
  3. 其他非本群组的权限

其中,[r]代表可读(read)、[w]代表可写(write)、[x]代表可执行(execute), [-]表示没有权限, 这三个权限的位置不会改变
